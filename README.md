# 技术栈

nodejs + express + mongodb + mongoose + es6/7

## 项目运行

```

git clone git@github.com:EasyTuan/node-elm-api.git

cd nuxt-elm

npm install

npm run dev

```

### 1.2 开发过程

#### 1.2.1 命令

```sh

# 开发
npm run dev

# 启动mongodb（确保已经安装了mongodb）
npm run mongod

# 开启浏览器GUI界面查看mongodb（默认端口1234）
npm run mongo

# 生成api文档（基于apidoc）
npm run doc

# 生产部署（pm2进程守护）
npm run start

```
